# Wiktor Ciepliński – Portfolio

My personal portfolio website built with **React** and **Tailwind CSS**.  
It showcases my skills, projects, and provides a contact section.

## 🚀 Features

- Responsive layout – works on mobile and desktop
- Hero section with name and description
- Skills section with categorized skills
- Projects section with links to GitHub repositories
- Contact section with links to social media
- Modern design with gradients, hover effects, and animations

## 🛠 Technologies Used

- React
- Tailwind CSS
- JavaScript
- HTML / CSS
- React Icons

## 📌 How to Run Locally

1. Clone the repository:
```bash
git clone https://github.com/Magnes77/portfolio.git

2. Install dependencies:
npm install

3. Run the development server:
npm run dev
# or
npm start

4. Open http://localhost:5173
 (Vite) in your browser.
 
 ```
